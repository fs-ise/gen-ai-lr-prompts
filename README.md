# GenAI Prompts for Literature Reviews

## Overview

This repository provides a **curated collection of prompts** designed to support **standalone literature review papers** using generative AI tools (e.g., ChatGPT and related systems).

The prompts are organized along the core stages of the literature review process and are grounded in academic research on the use of generative artificial intelligence in literature reviews. The repository builds on:

> Wagner, G., Prester, J., Mousavi, R., Lukyanenko, R., & ParÃ©, G. (2026). *Generative artificial intelligence for literature reviews*. (to be accepted at the *Journal of Information Technology*)

## Review Process Coverage

The repository follows the main stages of a literature review:

- **Problem formulation**
- **Search**
- **Screen**
- **Quality assessment**
- **Data extraction**
- **Data analysis**

## Scope

This repository focuses on:
- Prompts for **standalone literature review papers**
- Methodological support across the full review workflow
- Reusable and adaptable prompt designs

It does not aim to provide complete review automation or domain-specific reviews.
